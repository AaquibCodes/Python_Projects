This code generates a roll no. from the given registrattion number.

- It groups the registration number into groups, each group containing 2 index values of the registration no.
- The maximum value of each group is considered and appended to the rollno.
- If any value is not been grouped then it is added to rollno. by default.
- If the values in any group are same then it will take only value.

example - 
	Enter the registration number : 4536262
	Roll number generated is : 5662
